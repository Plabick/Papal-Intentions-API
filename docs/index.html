get /{year}/{month}
Returns the Holy Fathers Intention's for the specified month in English. (yearMonthGet)

Path parameters

year (required)
Path Parameter — A 4 digit number representing the year. Minimum 2020. default: null
month (required)
Path Parameter — The month to return the intention for. default: null

Return type

inline_response_200

Example data

Content-Type: application/json
{
  "name" : "Let us pray for young people who are preparing for marriage with the support of a Christian community: may they grow in love, with generosity, faithfulness and patience.",
  "title" : "The Beauty of Marriage"
}
